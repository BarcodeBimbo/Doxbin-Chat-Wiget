Doxbin Chat Snooper Widget

This page demonstrates the Doxbin Chat Snooper widget that can be embedded on any static HTML site.

How to Embed the Widget

To embed this widget on your static site, add the following iframe to your HTML:

<iframe src="http://your-server-address:8934/widget" width="100%" height="100%" frameborder="0" style="border:none;position:fixed;bottom:0;left:0;width:100%;height:100%;z-index:999;pointer-events:none;"></iframe>

The widget will appear in the bottom left corner of your website with a minimize/maximize button.

Important Notes:

  • The widget uses pointer-events:none on the iframe to allow users to interact with your website content. Only the widget itself can be clicked.
  • The widget position is fixed at the bottom left corner of the window.
  • The widget state (minimized/maximized) is saved in local storage.
